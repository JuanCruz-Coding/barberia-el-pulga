# Barbería "El Pulga" — Landing Page

Proyecto de portafolio: landing page para una barbería real.

## Datos del negocio

| Campo | Detalle |
|---|---|
| Nombre | Barbería "El Pulga" |
| Tipo | Barbería / Peluquería |
| Servicios | Corte de cabello, Corte de barba, Diseño de cejas, Afeitado clásico |
| Ciudad | Gobernador Gálvez, Santa Fe |

## Paleta de colores sugerida

Barbería clásica/masculina → tonos oscuros y dorados funcionan muy bien.

| Rol | Color | Hex |
|---|---|---|
| Fondo principal | Negro carbón | `#1a1a1a` |
| Acento / CTA | Dorado | `#c9a84c` |
| Texto principal | Blanco | `#f5f5f5` |
| Fondo secundario | Gris oscuro | `#2d2d2d` |

> Podés cambiar esta paleta si tu hermano prefiere algo más moderno/minimalista.

## Tipografía sugerida (Google Fonts, gratis)

- Títulos: **Playfair Display** (elegante, clásica)
- Cuerpo: **Inter** o **Lato** (limpia, legible)

## Estructura del sitio

```
index.html
├── Nav          → Logo + links de navegación
├── Hero         → Frase principal + botón "Reservar turno"
├── Servicios    → 4 tarjetas (corte, barba, cejas, afeitado) + precio
├── Sobre nosotros → Quiénes somos, experiencia
├── Galería      → Fotos del trabajo (opcional v1)
├── Contacto     → Formulario + WhatsApp + dirección
└── Footer       → Horarios, redes, © 2025
```

## Stack técnico

- HTML5 semántico
- CSS3 (variables, Flexbox, Grid, media queries)
- JavaScript vanilla (validación de formulario, menú hamburguesa, scroll suave)
- Deploy: Netlify (gratis)
